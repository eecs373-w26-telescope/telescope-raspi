# Telescope HUD using Raylib

## Dependencies

```bash
./scripts/deps.sh
```

Installs: cmake, build-essential, git, libdrm-dev, libgbm-dev, libinput-dev, libudev-dev

## Build

```bash
./scripts/build.sh
./scripts/build.sh deploy # for raspi
```

## Run

```bash
./cmake-build/373-telescope
sudo ./cmake-deploy/373-telescope
```

## Clean

```bash
./scripts/clean.sh
```
